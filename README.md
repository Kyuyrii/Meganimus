# Current situation of Meganimus
Currently, I can't continue developing Meganimus. Meganimus and what it does are simple, however, I've noticed that emulators commonly save specific paths instead of relative ones, which is the opposite of what Meganimus does.

So it's much more complicated to deal with emulators in Meganimus than with Wine/Proton/UMU in Zordeer.

It might seem dramatic that I'm stopping Meganimus development just because of this, but it's really something that has discouraged me.
